# Padrões de Projeto com PHP 8.1

### Passo a passo para rodar o projeto

Rodar o container

```sh
docker-compose up -d
```

Acessar o container
```sh
docker-compose exec app bash
```


Instalar as dependências do projeto
```sh
composer install
```


Acessar o projeto
[http://localhost:8989](http://localhost:8989)
